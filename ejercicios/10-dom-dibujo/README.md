# 10. DOM basico - dibujo

## 🟦 Objetivo

Modificar una tarjeta de dibujo desde JavaScript.

## 🟩 Instrucciones

Abre `ejercicio.js` y prueba el resultado desde `index.html` en el navegador. selecciona elementos del HTML y cambia texto, color y clases.

## 🟨 Pistas

- Divide el problema en pasos pequenos.
- Usa `console.log` para revisar valores intermedios.
- Revisa el nombre exacto de variables, funciones y selectores.

## 🟥 Reglas

- No copies codigo desde `respuestas/` antes de intentarlo.
- No uses frameworks ni librerias externas.
- Mantén la solucion en `ejercicio.js` salvo que el enunciado pida HTML o CSS.
- Usa nombres descriptivos y codigo ordenado.

## 🟪 Reto extra

Agrega una validacion adicional o un dato extra relacionado con dibujo.

## ✅ Resultado esperado

El programa debe mostrar un resultado claro y coherente con el tema de dibujo, sin errores en consola.

## Conceptos que practica

- querySelector
- textContent
- style
- classList
