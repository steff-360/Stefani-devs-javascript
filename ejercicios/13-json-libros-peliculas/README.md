# 13. JSON - libros y peliculas

## 🟦 Objetivo

Convertir informacion entre objetos JavaScript y JSON.

## 🟩 Instrucciones

Abre `ejercicio.js` y ejecutalo con Node.js o desde la consola del navegador. serializa y parsea una lista de libros y peliculas favoritas.

## 🟨 Pistas

- Divide el problema en pasos pequenos.
- Usa `console.log` para revisar valores intermedios.
- Revisa el nombre exacto de variables, funciones y selectores.

## 🟥 Reglas

- No copies codigo desde `respuestas/` antes de intentarlo.
- No uses frameworks ni librerias externas.
- Mantén la solucion en `ejercicio.js` salvo que el enunciado pida HTML o CSS.
- Usa nombres descriptivos y codigo ordenado.

## 🟪 Reto extra

Agrega una validacion adicional o un dato extra relacionado con libros y peliculas.

## ✅ Resultado esperado

El programa debe mostrar un resultado claro y coherente con el tema de libros y peliculas, sin errores en consola.

## Conceptos que practica

- JSON.stringify
- JSON.parse
- arrays de objetos
