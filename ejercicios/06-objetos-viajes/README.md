# 06. Objetos - viajes

## 🟦 Objetivo

Representar un viaje con propiedades y metodos simples.

## 🟩 Instrucciones

Abre `ejercicio.js` y ejecutalo con Node.js o desde la consola del navegador. crea un objeto viaje con destino, dias, presupuesto y metodo resumen.

## 🟨 Pistas

- Divide el problema en pasos pequenos.
- Usa `console.log` para revisar valores intermedios.
- Revisa el nombre exacto de variables, funciones y selectores.

## 🟥 Reglas

- No copies codigo desde `respuestas/` antes de intentarlo.
- No uses frameworks ni librerias externas.
- Mantén la solucion en `ejercicio.js` salvo que el enunciado pida HTML o CSS.
- Usa nombres descriptivos y codigo ordenado.

## 🟪 Reto extra

Agrega una validacion adicional o un dato extra relacionado con viajes.

## ✅ Resultado esperado

El programa debe mostrar un resultado claro y coherente con el tema de viajes, sin errores en consola.

## Conceptos que practica

- objetos
- propiedades
- metodos
- this
